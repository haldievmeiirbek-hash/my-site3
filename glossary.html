Глоссарий

Python терминдер сөздігі

🔍
📦
Айнымалы
Variable
Мәліметтерді сақтауға арналған атауы бар жад ұяшығы. Python-да: x = 10
🔄
Цикл
Loop
Бір немесе бірнеше командалардың белгілі бір шарт орындалғанша немесе белгіленген рет қайталануы. for, while операторлары қолданылады.
🔀
Шарт
Condition
Бағдарламаның белгілі бір жағдайға байланысты таңдау жасауы. Python-да if-elif-else операторы арқылы іске асырылады.
⚙️
Функция
Function
Нақты бір тапсырманы орындайтын атауы бар код бөлігі. def кілт сөзімен анықталады.
📋
Алгоритм
Algorithm
Есепті шешу үшін орындалатын қадамдардың нақты реттелген тізімі. Дискреттілік, анықтылық, нәтижелілік қасиеттері бар.
🏷️
Деректер типі
Data Type
Айнымалының сақтайтын мәліметінің түрі. Python-да: int (бүтін), float (нақты), str (мәтін), bool (логикалық).
➕
Оператор
Operator
Мәліметтермен амалдар жасауға арналған арнайы белгі: +, -, *, /, **, %, //, ==, !=, <, > және т.б.
🐛
Синтаксис қатесі
Syntax Error
Python тілінің жазылу ережелерін бұзған кезде пайда болатын қате. Бағдарлама іске қосылмайды.
