🐍 1. Python тілінің негіздері

Python — 1991 жылы Гвидо ван Россум жасаған жоғары деңгейлі программалау тілі. Оның синтаксисі табиғи тілге жақын, сондықтан үйрену оңай.

💡 Неліктен Python? Бүгінде Python Data Science, Sun Intelligence, Веб-разработка, автоматизация сияқты күрделі саларда кең қолданылады. Дүниежүзіндегі ең танымал тіл.

Python бағдарламасының құрылымы

# Бұл — комментарий (Python орындамайды) # Бағдарламаның аты: Алғашқы бағдарлама print("Сәлем, Қазақстан!") # экранға шығару print("Python үйренемін 🐍")

print() функциясы — деректерді шығару

print("Мәтін") # жолды шығару print(42) # санды шығару print("Сан:", 42) # бірнеше мән print("а =", 5, "+", 3, "=", 5+3) print() # бос жол print("Бітті!", end="") # жол аудармай

input() функциясы — деректерді енгізу

аты = input("Атыңызды енгізіңіз: ") print("Сәлем,", аты, "!") жас = int(input("Жасыңыз: ")) # санға айналдыру print("Сіз", жас, "жастасыз")

➕ 3. Операторлар

Арифметикалық операторлар

a + b # қосу a - b # азайту a * b # көбейту a / b # бөлу (float) a // b # бүтін бөлу a % b # қалдықты табу a ** b # дәрежеге

Салыстыру операторлары

a == b # тең a != b # тең емес a > b # үлкен a < b # кіші a >= b # үлкен немесе тең a <= b # кіші немесе тең

Логикалық операторлар

and # екеуі де True or # кем дегенде бірі not # керісінше x > 0 and x < 10 y == 0 or y == 5

Меншіктеу операторлары

x = 5 x += 3 # x = x + 3 = 8 x -= 2 # x = x - 2 = 6 x *= 2 # x = x * 2 = 12 x //= 4 # x = x // 4 = 3

Практикалық мысал

a, b = 17, 5 print(f"{a} + {b} = {a+b}") print(f"{a} // {b} = {a//b}") # бүтін бөлу: 3 print(f"{a} % {b} = {a%b}") # қалдық: 2 print(f"2^10 = {2**10}") # 1024 # Математика модулі import math print(math.sqrt(16)) # 4.0 print(math.pi) # 3.14159... print(math.factorial(5)) # 120

🔀 4. Тармақталған алгоритм (if-elif-else)

Тармақталған алгоритм — бағдарлама белгілі шартқа байланысты әртүрлі жолмен орындалады.

Бастау
↓
Шарт тексеру
↙ Ия     Жоқ ↘
if блогы орындалады
else блогы орындалады
↓
Аяқтау

if-elif-else синтаксисі

жас = 15 if жас < 7: print("Мектепке дейінгі") elif жас <= 11: print("Бастауыш мектеп") elif жас <= 15: print("Орта буын") elif жас <= 18: print("Жоғары сынып") else: print("Ересек")

Күрделі шарттар

x = 7 # and — екі шарт бірдей орындалуы керек if x > 0 and x < 10: print("0-ден 10-ға дейін") # or — кем дегенде бір шарт if x == 0 or x == 7: print("Нөл немесе жеті") # Ішкі (nested) if баға = 85 if баға >= 90: print("A — Үздік") elif баға >= 75: print("B — Жақсы") elif баға >= 50: print("C — Қанағаттанарлық") else: print("F — Қайта тапсыру") 💪 Тапсырмаларды орында

🔄 5. Циклдер (for және while)

Цикл — бір немесе бірнеше операторды белгілі рет немесе шарт орындалғанша қайталайды.

For цикл — белгілі рет қайталау

# range() — сан тізбегі жасайды for i in range(5): print(i) # 0, 1, 2, 3, 4 for i in range(1, 6): print(i) # 1, 2, 3, 4, 5 for i in range(0, 20, 5): print(i) # 0, 5, 10, 15 (қадам=5) for i in range(10, 0, -1): print(i) # 10-нан 1-ге кері санау

Тізіммен жұмыс

жемістер = ["алма", "алмұрт", "жүзім", "банан"] for жеміс in жемістер: print(f"{жеміс} — дәмді!") # enumerate — индекс пен мән бірге for i, жеміс in enumerate(жемістер, 1): print(f"{i}. {жеміс}")

While цикл — шарт орындалғанша

n = 1 while n <= 5: print(f"{n} × {n} = {n*n}") n += 1 # break және continue for i in range(10): if i == 3: continue # 3-ті өткізіп жіберу if i == 7: break # 7-де тоқтату print(i) # 0,1,2,4,5,6

Кірістірілген циклдер (nested loops)

# Жұлдыз үлгісі for i in range(1, 6): print("★" * i) # Көбейту кестесі for i in range(1, 4): for j in range(1, 4): print(f"{i}×{j}={i*j}", end=" ") print() # жаңа жол 💪 Практикада қолдан

⚙️ 6. Функциялар (def)

Функция — бір немесе бірнеше операцияны атаулы блокқа топтастырады. Кодты қайталамауға (DRY принципі) мүмкіндік береді.

Функция анықтау және шақыру

# Анықтау def сәлемдесу(аты): print(f"Сәлем, {аты}! 👋") # Шақыру сәлемдесу("Айгерім") сәлемдесу("Данияр") сәлемдесу("Нұрлан")

Мән қайтаратын функция (return)

def аудан_есептеу(ұзындық, ені): аудан = ұзындық * ені return аудан def максимум(a, b): if a > b: return a return b нәтиже = аудан_есептеу(5, 3) print(f"Ауданы: {нәтиже} м²") print(f"Максимум: {максимум(12, 7)}")

Әдепкі мән (default parameter)

def таныстыру(аты, жас=14, қала="Алматы"): print(f"{аты}, {жас} жас, {қала}") таныстыру("Айгерім") # Айгерім, 14 жас, Алматы таныстыру("Данияр", 15) # Данияр, 15 жас, Алматы таныстыру("Нұрлан", 13, "Астана") # Нұрлан, 13 жас, Астана

Нұсқаулар тобы белгілі рет немесе шарт орындалғанша қайталанады.

🧩 Рекурсивті алгоритм

Функция өзін-өзі шақырады. Factorial, Fibonacci есептеу.

Тәжірибелік алгоритм: Максималды санды табу

  • Бастапқы максимумды бірінші элементке тең деп аламыз
  • Тізімдегі әр элементті максимуммен салыстырамыз
  • Егер элемент максимумнан үлкен болса — жаңа максимум деп саламыз
  • Тізімнің соңына жеткенде максимумды шығарамыз
def максималды_табу(тізім): макс = тізім[0] for сан in тізім: if сан > макс: макс = сан return макс сандар = [3, 17, 8, 42, 5, 1] print(f"Максималды сан: {максималды_табу(сандар)}") # 42

Іздеу алгоритмі (Linear Search)

def сызықтық_іздеу(тізім, мақсат): for i, мән in enumerate(тізім): if мән == мақсат: return i # индексін қайтару return -1 # табылмаса сандар = [10, 25, 3, 47, 8] нәтиже = сызықтық_іздеу(сандар, 47) print(f"47 саны {нәтиже}-індексте орналасқан") # 3 💪 Тапсырмаларды орында
